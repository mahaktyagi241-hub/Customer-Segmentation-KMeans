# Customer Segmentation using K-Means Clustering

## Project Overview
This project uses the K-Means Clustering algorithm to segment mall customers based on their Annual Income and Spending Score. Customer segmentation helps businesses understand customer behavior and create targeted marketing strategies.

---

## Objective
To divide customers into different groups using Machine Learning so that businesses can identify different customer types.

---

## Dataset
- Dataset: Mall Customers Dataset
- Total Records: 200
- Features:
  - CustomerID
  - Gender
  - Age
  - Annual Income (k$)
  - Spending Score (1-100)

---

## Technologies Used

- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn

---

## Machine Learning Algorithm

- K-Means Clustering

---

## Steps Performed

1. Imported libraries
2. Loaded dataset
3. Performed data exploration
4. Selected features
5. Used Elbow Method to find optimal clusters
6. Applied K-Means Clustering
7. Visualized customer segments
8. Saved clustered dataset

---

## Output

The customers were divided into **5 different clusters** based on Annual Income and Spending Score.

## Conclusion

The K-Means algorithm successfully segmented customers into different groups. These customer segments can help businesses improve marketing strategies and customer targeting.
